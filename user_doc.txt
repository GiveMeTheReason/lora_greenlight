Инструкция для пользователя:

Изначально доступные команды:
/help - получить помощь
/start - начать работу с ботом

Для начала работы введите команду "/start"
Затем отправьте ваш логин (e-mail) боту
Затем отправьте ваш пароль боту
(После отправки, сообщение с паролем будет удалено)

При успешной авторизации бот об этом сообщит
При неудачной авторизации бот попросит ввести логин еще раз

После авторизации доступны команды:
/help - получить инструкцию
/start - проверка авторизации
/exit - выход из аккаунта, для новой авторизации придется вводить пароль еще раз

После авторизации нужно получить список шкафов ШУНО при помощи кнопки:
"Получить список ШУНО"

После получения списка шкафов, выберите нужный при помощи кнопок

Теперь вы можете отправлять фотографии с QR-кодами в виде файлов боту
Все фотографии будут отнесены к выбранному шкафу ШУНО

Фотографии должны содержать четкие и контрастные изображение QR-кода, без отсвечивания и размытостей
Фотографии должны содержать метку о геолокации полученную при помощи функций смартфона или других приложений
Максимальный размер загружаемой фотографии: 20 Мб (мегабайт) (этот параметр можно изменить при запуске)

При неудачном распозновании QR-кода бот уведомит пользователя
При успешном распозновании QR-кода бот сообщит прочитанную информацию

Чтобы сменить выбранный шкаф ШУНО нажмите на кнопку:
"Получить список ШУНО"
Если кнопка "пропала" введите "Получить список ШУНО" вручную (регистр не важен)
